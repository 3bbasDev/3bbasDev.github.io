01 / EDGE

Where users meet the machine.

Real-time WebSocket services and clean REST surfaces push live state to clients the instant it changes — no refresh, no polling, no waiting.

02 / GATEWAY

One door, guarded well.

ASP.NET Core gateways handle auth, role-based permissions, and tenant isolation — so each request lands exactly where it should, with exactly the access it's allowed.

03 / SERVICES

Small services, big systems.

Independent C# and Golang microservices, each owning its domain and talking over gRPC and well-defined contracts. Deploy one without waking the others.

04 / DATA

Memory that scales.

PostgreSQL for truth, Redis for speed, Elasticsearch for search. Structured for fast reads, honest writes, and queries that return before you blink.

05 / INTEGRATIONS

Where money & government meet.

Payment gateways, POS, bank integrations and e-government links — the hardest, highest-stakes part of every platform I've shipped, from e-invoicing to national transport.
